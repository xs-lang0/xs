# Welcome to My Site

This is a **static site** built with XS.

## Features

The generator handles *basic markdown* and turns it into clean HTML.

Here's what it supports:

- Headings with `#` and `##`
- **Bold** and *italic* text
- Inline `code` snippets
- Horizontal rules
- Paragraph breaks

---

Built with the XS static site generator. Simple, fast, no dependencies.
