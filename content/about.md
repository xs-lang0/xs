# About

This site was generated from **markdown-ish** files using a static site generator written in XS.

## How It Works

The generator reads `.md` files from `content/`, converts basic formatting to HTML, wraps everything in a template with styles, and writes the result to `build/`.

It's about **100 lines** of XS code. No external dependencies.

## The XS Language

XS is a small, expressive scripting language with:

- First-class functions and closures
- Pattern matching
- A clean `import` system for modules
- Built-in `fs` and `path` modules for file I/O

---

*That's it. Nothing fancy.*
